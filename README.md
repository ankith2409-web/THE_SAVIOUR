🚨 Emergency Healthcare Response System A real-time emergency healthcare management platform built with React, TypeScript, and Firebase — connecting patients, hospitals, and police for faster emergency response.

🌟 Features

🏥 Hospital Portal — Manage incoming emergency cases and patient records 🧑‍⚕️ Patient Portal — Report emergencies and track response status in real-time 🚔 Police Portal — Coordinate with hospitals during emergency incidents 🤖 Emergency Bot — AI-powered bot to guide users during emergencies 🧠 Medical AI — Smart medical assistance and triage suggestions 📍 Location Tracker — Real-time location tracking for emergency dispatch 🔔 Notification Service — Instant alerts to relevant parties 🔄 Sync Service — Real-time data sync across all portals using Firestore
